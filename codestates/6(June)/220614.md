주어진 배열에서 arr[i]값은  
이전의 모든 수의 합 보다 커야한다
```java
package com.codestates.coplit; 
import java.util.*;

public class Solution { 
	public boolean superIncreasing(int[] arr) {
		boolean answer = true;
        int sum = arr[0];
        
        for (int i = 1; i < arr.length-1; i++) {
            sum+=arr[i];
            answer = answer&&(sum<arr[i+1]);
        }
        
        return answer;
	} 
}
```  
### 풀이  

반복문으로 각 수를 더함과 동시에 비교한 값을 결과 boolean에 반영했다.
