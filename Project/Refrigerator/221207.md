### 피드백
친구들에게 피드백을 받아봤다.  
추가, 수정해야할 부분들을 알게 되었다.
- 칸 순서 조정기능
    - 중요도가 높은 칸이 나중에 생성되는 경우, 순서를 수정할 수가 없다.  
    프론트에 좀 더 가까울것같긴한데, 즐겨찾기 등으로 우선순위를 조정할 수 있을 것 같다.
- 회원가입 유효성검사
    - 비밀번호 확인 추가 : 비밀번호 입력 실수를 방지하기 위한 기능

    - 비밀번호 조건 추가
